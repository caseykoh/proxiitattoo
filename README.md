# Proxii World

## Project Description
A React application that allows users to book tattoo appointments from the artist proxii_dream, view portfolio work, and gain information on the artist's booking process.
You can check out the live version [here](https://proxiiworld.com/).

## Technologies Used
- TypeScript
- React
- Vite
- Axios
- CSS

## Installation Instructions
```bash
git clone https://github.com/yourusername/repo-name.git
cd repo-name
npm install
npm start
```

## Usage
Navigate to http://localhost:5173/ to view the application.
